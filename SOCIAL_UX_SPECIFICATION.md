# KAYFABE Social Features - UX/UI Specification

**Date:** 2026-01-17
**Purpose:** Detailed user experience design for all social/multiplayer features
**Status:** Design Phase - Pending Review

---

## Design Principles

1. **Wrestling Authenticity** - Every feature feels like a wrestling promotion (kayfabe maintained)
2. **Progressive Unlocking** - Features reveal themselves as users gain XP (creates "aha!" moments)
3. **Non-Intrusive Social** - Multiplayer features are opt-in, never interrupt single-player flow
4. **Visual Hierarchy** - Gold accents for unlocked features, gray for locked features
5. **Mobile-First** - All interactions work seamlessly on phone screens

## Notifications System

### Achievement Unlocks
- Custom name unlocked (250 XP)
- Gimmick unlocked (10,000 XP)
- Full edit access (25,000 XP)
- New avatar unlocked

### Challenge Notifications
- Challenge received
- Challenge accepted/declined
- Opponent made progress
- Challenge won/lost
- Rematch requested

### Faction Notifications
- Invited to faction
- Faction request accepted/declined
- Promoted to leader
- Faction reached new rank

## Mobile Responsiveness

All screens designed mobile-first:
- Single column layouts
- Large touch targets (min 44px)
- Swipe gestures for tabs
- Bottom navigation for main tabs
- Modals use full screen on mobile

---

## Accessibility

- High contrast (gold on black, cream on black)
- Large text sizes
- Screen reader support
- Keyboard navigation
- Focus indicators
- Color is not sole indicator (emojis + text)

## Questions for Review

1. **Progressive Identity:**
   - Should gimmicks be editable after first set, or locked like ring names?
   - Should we show a preview of locked features earlier (motivational)?

2. **Leaderboards:**
   - Real-time updates or refresh intervals?
   - Show full 100 or just Top 10 + user rank?

3. **Challenges:**
   - Should challenges have entry costs (XP)?
   - Allow best-of-3 series?

4. **Factions:**
   - Maximum faction size (50 members)?
   - Should factions have ranks/permissions beyond leader?
   - Faction-only chat needed?

5. **General:**
   - Push notifications for challenges/factions?
   - Dark mode vs light mode options?
   - Sound effects for achievements?
